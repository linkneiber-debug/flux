# ⚡ Flux Launcher

A modern, high-performance game launcher desktop application inspired by Project Stellar. Built with Electron and React, Flux Launcher provides a sleek, intuitive interface for managing and launching your game library.

## 🌟 Features

- **Modern UI Design**: Stunning gradient backgrounds with neon accents and smooth animations
- **Cross-Platform**: Runs on Windows, macOS, and Linux
- **Game Library Management**: Organize and manage your installed games
- **Featured Games Section**: Discover and highlight new games
- **User Profile**: Integrated user profile and status system
- **Responsive Design**: Adapts to different screen sizes
- **Dark Theme**: Eye-friendly dark mode by default
- **Quick Actions**: Play, install, and configure games with one click
